# sistema-biblioteca-flask

## Como executar em Desenvolvimento

```bash
make build-dev
```
```bash
make bash
```
```bash
flask db upgrade
```
```bash
flask run --host=0.0.0.0
```
ou
```bash
make flask-run
```
